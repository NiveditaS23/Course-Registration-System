# Course-Registration-System
This is a Flask-based Course Registration System project. It allows users to:
1. Login and logout with predefined credentials.
2. View a list of available courses categorized into Science, Arts, and Commerce.
3. Register students to courses.
4. Edit existing registrations.
5. Delete registrations with confirmation.
6. Search/filter registered students by name.
The UI is modern, responsive, and color-coded for easy identification of course categories.
